量化噪声

信号的功率

信号总是伴随着噪声，所谓 Sigma-Delta 调制器（ Sigma-Delta modulator ，SDM ）是一种将低频噪声搬移到高频的调制技术，这样只要经过低通滤波就能滤除噪声，还原出有用信号，因而广泛地应用在 ADC 和 DAC 中

以 DAC 为例，即使输入的数字信号是足够优质，几乎没有噪声的，在数字-模拟转换的过程中也会引入噪声，称为量化噪声（ quantization noise ）。因为 DAC 的输出一定是折线形不连续的，就会出现和原信号间的误差，这就是量化噪声

信号功率和噪声功率的比值称为信噪比（ Signal Noise Ratio ，SNR ），这个值越高说明信号相对于噪声的功率越大，噪声的干扰效果越微弱，信号能量和功率的定义是

过采样

白噪声是指噪声的频谱在整个频域中均匀分布，这个词来源于光学，当各种频率的光均匀分布时就得到白光。而有色噪声则相反，指某一频段的噪声幅值明显比其它频段大，当低频噪声比较显著时称为红噪声，而高频噪声比较显著时称为蓝噪声。量化噪声一般可以视为白噪声，均匀分布在各个频率下

接下来 Nyquist 采样定理表明对于带宽为 的信号，采样频率至少要在 才能保留原信号携带的信息，实际中许多时候会采取比 更高的频率来采样，称为过采样。过采样并不难发生，特别是对于音频信号，其频率在 20Hz 到 20kHz 之间，而现代 ADC 和处理器时钟频率随便都能上 MHz 级别

在过采样的情况下，频谱宽度 大于信号带宽 ，这样比 更高的高频区域就只有量化噪声，而没有有用信号。而 Fourier 变换理论中的 Parseval 等式表明，信号频谱幅值的平方关于频域求积分同样是信号的能量，最多相差一个 的系数，即

功率也与频谱幅值的平方有相似的关系。所以如果过采样之后，再经过一个低通滤波器将高频噪声滤掉，噪声的能量/功率就会降低，从而大大提高信噪比

更有甚者，如果有某种方法能将低频噪声搬移到高频去，使白/红噪声成为蓝噪声，再进行低通滤波就会得到更好的效果

脉冲密度调制 ( PDM )

脉冲密度调制，顾名思义是将输入信号的幅值大小用输出信号的脉冲密度来表示。比如下图中输入是正弦信号，在输入信号为比较大的正值的时候输出的 1（正脉冲）比较多，而输入信号为比较大的负值的时候输出的 0（负脉冲）比较多，而输入信号接近零的时候输出不断在正负脉冲之间切换，所以看起来非常密集

PDM 具体的实现方法不止一种，最为常见的实现方法是所谓脉冲宽度调制（ pulse width modulation ，PWM ）

脉冲宽度调制 ( PWM )

PWM 的做法是先确定一个时间窗口 ，在这个时间窗口内按照输入的幅值确定输出脉冲的占空比

假定输入是一个 4 位有符号数，从 到 一共有 16 个数，那么就把这个时间窗口分为 16 个“子窗口”。规定当输入为 -8 时所有子窗口内均输出 0 ，而每当输入加 1 ，就多一个子窗口输出 1 ；比如输入为 2 时就有 10 个子窗口输出 1 ，而有 5 个子窗口输出 0 ，而输入为 7 时则有 15 个子窗口输出 1 ，1 个子窗口输出 0

但是上述叙述有个前提条件：信号在整个时间窗口内不变，或者说时间窗口即为信号的采样周期。但是这个前提并不是总能满足，有时在一个时间窗口内信号的值也会变化

这时一个简单的做法是构造一个计数器，当刚进入一个时间窗口时将其置为最小值，然后每过一个子窗口的时间对其加 1 ，这样在该时间窗口结束时它刚好加到最大值。同时，对输入信号的值和计数器的值进行比较，如果输入信号大于计数器则输出 1 ，反之输出 0

FPGA 实现 1-bit SDM-DAC

完整的 1-bit SDM-DAC 包含两部分：

  • 数字部分：数字 SDM ，也是 DAC 的主体，SDM 接收输入的数字信号，并将其转为 PDM 信号输出

  • 模拟部分：模拟 LPF ，接收 PDM 信号，并滤除其中的高频噪声，得到模拟信号

FPGA 可以实现的是数字部分
